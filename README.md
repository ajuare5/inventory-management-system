# Inventory Management System

A Python-based inventory management system built with object-oriented design. Supports product tracking, transaction processing, stock updates, and inventory reporting through a modular, extensible architecture.

## Author
Arturo Juarez Jr.

## Features
- Add, remove, and update inventory items
- Process sales and restock transactions
- Track inventory levels and generate reports
- Persistent storage via text file I/O
- Modular design with separate classes for inventory items and transactions

## Repository Structure
```
├── main.py                         # Main entry point and program logic
├── functions.py                    # Core helper functions
├── inventory.py                    # Inventory item class definition
├── transactionitem.py              # Transaction class definition
└── Inventory.txt                   # Sample inventory data file
```

## Technologies
- Python 3 (OOP, file I/O, data structures)
